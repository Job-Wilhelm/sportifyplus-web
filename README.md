# Sportify+ 前端環境

## 安裝套件
```
// 透過 package-lock.json 安裝版本相符的套件
$ npm ci 
```

## 專案指令
```
// 開啟本地伺服器
$ npm run dev

// 打包專案
$ npm run build

// 預覽打包後專案內容
$ npm run preview
```
